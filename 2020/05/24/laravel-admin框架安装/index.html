    字数:151 阅读 :
    1. laravel-admin框架安装

    laravel-admin框架安装

    1. composer create-project --prefer-dist laravel/laravel blog

    2. 修改根目录下的.env配置文件，修改相应数据库连接

    3. composer require encore/laravel-admin

    4. 发布资源：php artisan vendor:publish --provider="Encore\Admin\AdminServiceProvider"

    5. 修改数据库配置：\config\database.php

      'charset' => 'utf8',
'collation' => 'utf8_unicode_ci',
'prefix' => '',
'strict' => true,
'engine' => 'InnoDB'

    6. 创建表：php artisan admin:install

    7. disk配置：在\config/filesystems.php中加入

      'admin' => [
            'driver' => 'local',
            'root' => public_path('upload'),
            'visibility' => 'public',
            'url' => env('APP_URL').'/public/puload/',
            ]

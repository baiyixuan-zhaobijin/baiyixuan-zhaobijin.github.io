    • 老磁带

    笔记
    1. 那些老磁带，到底是心头之肉，肉和筋连在了一起，老磁带是肉，光阴是筋，如何也分扯不开呀。
    2. 乌溜溜的黑眼珠和你的笑脸，怎么也难忘记你容颜的转变。
    书纪
    1. 曾几何时，当你回忆起红尘相遇的爱情，伤感与无奈便似决了堤的洪水把你淹没。但当你继续回忆，点上一支烟，泡上一杯茶，燃上一段香，所有的黯淡与萧瑟便如春风化雨，春雨潇潇，静静滋养你的温柔与醇厚。爱上，爱过，别离……仿佛是流光的三部曲，让生命漫溢着朝阳的华彩，炎阳的燃灼以及夕阳的妖娆，所有的相遇都是生命的奇迹，而我爱你，便是奇迹里至盛大的美丽，于是，在未来的日子里，我都愿为你盛装而行。
    2. 没有回应，爱情成了绝望的前行，不知黎明在哪里，更不知你会在前路的哪个小客栈里给我继续前行的相思醉，只是爱你，早已成了生命里最深沉的执着，贯穿了岁月的始终，更贯穿着灵魂的所有。而现在，我唯有前行，带着昨夜的嫣然一笑。
    3. 太多的时候，当我以为人书俱老，烟尘俱散的时候，心底里总有一份执着似无形的脊梁坚挺着，也许，这便是我生命的本源，哪怕所有的人与事都离我远去，我至少还有自己，坚执着去做一面迎风而立的旌旗，然后在红尘浮沉中千疮百孔，却始终不肯倒下。

    胭脂

    笔记
    1. 海棠着雨胭脂透，这是关于离散的爱情。
    2. 感情泛滥萌芽时，不涂胭脂也会飞粉面—恋爱中的女子，总会脸上有红云。她自己俨然不知，她只是一往情深地爱着他。
    3. 她粉面，说话亦粉嫩–小纤腰温柔无力，细碎的长发弯曲着，两道柳叶眉生动着，芬芳着。
    4. 没有原则地爱一个人，她早就变成他心里的一片最红的胭脂，染红了他的心。
    书纪
    1. 爱你，从来不需要理由，哪怕你与我早已远隔天涯，三生遥望。每一次想你，都是幸福的奏鸣曲在回荡，化作春风，化作雨露，化作岁月里所有的甜蜜与沉醉，而未来，我依旧念你，如时光亲吻了我的皱纹。
    2. 这场生命，你是唯一的奇迹，如神把幸福交付在我的手里，只是这深烈的爱让你我远隔天涯，这大抵就是爱的罪，哪怕身处炼狱，也要给你带去片刻的幸福。
    3. 原来这么些年，我始终都在低头喝闷酒，醉的泪眼迷离，醉的人事不知，而今天，现在，我愿轻拂尘垢，让生命燃点光亮，便如时光给予我所有光明的馈赠。
    4. 所以，你一定要幸福哦，不然我会不开心的啦。
